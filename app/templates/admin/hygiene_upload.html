Higienizar Base de Leads

Esta ferramenta permite remover leads da base de dados em massa. Envie uma planilha (.csv ou .xlsx) que contenha uma coluna com os CPFs dos leads a serem removidos.

O sistema irá comparar os CPFs do seu ficheiro com os existentes na base de dados e irá apresentar uma lista para confirmação antes de apagar permanentemente os registos.

Apenas ficheiros .csv e .xlsx são permitidos.
